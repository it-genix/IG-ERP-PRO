# IG-ERP (Enterprise WordPress ERP Plugin)

A professional, scalable WordPress ERP plugin for Employee Management, Attendance, GPS Tracking, and Leave Management.

## 🚀 Installation

1. Download the `ig-erp` folder.
2. Zip the folder: `zip -r ig-erp.zip ig-erp/`
3. Upload to your WordPress site: **Plugins > Add New > Upload Plugin**.
4. Activate the plugin.

## 🛠 Features

- **Employee Management**: Manage roles, departments, and metadata.
- **Attendance System**: GPS-based check-in/out.
- **Geo Tracking**: Live map for employee locations.
- **Leave Management**: Request and approval workflow.
- **PWA Ready**: Can be installed on mobile devices.
- **REST API**: Integrated endpoints for mobile apps.

## 📁 Structure

- `admin/`: Backend admin pages (Dashboard, Employees, etc.)
- `includes/`: OOP Core logic and functional modules.
- `database/`: Custom database schema.
- `assets/`: Javascript (Leaflet maps, AJAX) and CSS.
- `templates/`: Frontend and admin views.
- `pwa/`: Manifest and Service Worker.

## 🔒 Security

- Nonce validation on all AJAX calls.
- Schema validation via `dbDelta`.
- Sanitized inputs and escaped outputs.
- Role-based access control (RBAC).
